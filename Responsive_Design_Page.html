Flexbox

Declaration

Value/Meaning
display: flex Turns an element into a flex container.
display: inline-flex Turns an element into a flex container that acts like an inline element.
justify-content:
  • flex-start (1st element at the left with no space)
  • flex-end (last element at the right with no space)
  • center (every elements are centered with no space between)
  • space-around (every elements are centered with space before and after each)
  • space-between (every elements are centered with space between but not around)
align-items:
  • flex-start (display items at the top of the container)
  • flex-end (display items at the bottom of the container)
  • center (display items at the vertical center of the container)
  • baseline (alligns the bottom of the items)
  • stretch (default value: occupies all space vertically)
align-self Same as align-item but for one element.
flex-grow: (enter numeric value) Allows an item to grow proportionally as its container and other content item's values (default is 0).
flex-shrink: (enter numeric value) Allows an item to shrink proportionally as its container and other content item's values (default is 1).
flex-basis: Sets a basic width to an item.
flex: "grow value" "shrink value" "basis value" Sets all three previous datas in one line.
flex-wrap:
  • wrap (child elements that don't fit into a row will move down to the next line)
  • wrap-reverse (same as wrap but the lines are inverted)
  • nowrap (default)
align-content:
  • flex-start (rows on top with no space)
  • flex-end (rows on bottom with no space)
  • center (rows centered with no space between)
  • space-between (rows on top and bottom with space between but not above or below)
  • space-around (rows spaced evenly from top and bottom with same amount between each)
  • stretch (default, stretches to fill container)
flex-direction:
  • row (default: positions elements from left to right starting from the top left corner)
  • row-reverse (positions elements from right to left starting from the top right corner)
  • column (positions elements from top to bottom starting from the top left corner)
  • column-reverse (positions elements from bottom to top starting from the bottom left corner)
flex-flow: (wrap direction) Declares both the flex-wrap and flex-direction properties in one line.
order: (integer) default is 0

Grid

Declaration

Value/Meaning
display: grid Turns on block level grid.
display: inline-grid Turns on inline grid
grid-template-columns: "value 1" "value 2"... sets the number of columns by writing px or % values.
grid-template-rows: "value 1" "value 2"... sets the number of rows by writing px or % values.
grid-template: (rows values)/(columns values)
(value)fr Sets the grid's length and with as fractions.
repeat("number of repeats", "value") Repeats what's indicated for rows or columns.
minmax("min value", "max value") Sets the min and max size of row or column.
row/column-gap: Add blank space between rows and columns.
gap: "row value" "column value" Adds the two gap values in one line.
grid-row/column-start/end: Allows an item to span accross multiple rows and/or columns(2 is 1 to 3 ; 4 is 3 to 7... or just use "span" keyword).
grid-row/column: "start value"/"end value" Shorthand for the previous declarations.
grid-area: "row-start value"/"column-start value"/"row-end value"/"column-end value" Shorthand por the previous declarations.
grid-template-area: "values" Refplaces start, end and area by section names.
justify-items:
  • start (align items on left)
  • end (align items on right)
  • center (centers items)
  • stretch (stretch items to fill the grid area)
justify-content:
  • start (align grid on left)
  • end (align grid on right)
  • center (centers grid horizontally)
  • stretch (stretches items to fill the grid horizontally)
  • space-around (equal space around items)
  • space-between (space between items but not on sides of grid)
  • space-evenly (equal space between items and at either end)
align-items:
  • start (align items to the top side)
  • end (align items to the bottom side)
  • center (align items to the center vertically)
  • stretch (stretches to fill grid area vertically)
align-content:
  • start (align grid on top)
  • end (align grid on bottom)
  • center (centers grid vertically)
  • stretch (stretches items to fill the grid vertically)
  • space-around (equal space around items)
  • space-between (space between items but not on ends of grid)
  • space-evenly (equal space between items and at either end)
justify/align-self
  • start (align items to the left/top side)
  • end (align items to the right/bottom side)
  • center (align items to the center horizontally/vertically)
  • stretch (stretches to fill grid area horizontally/vertically)
grid-auto-rows/columns: specifies the height/width of automatically added rows/columns.
grid-auto-flow:
  • row (new elements fill rows left to right and add rows if necessary)
  • column (new elements fill columns top to bottom and add columns if necessary)
  • dense (fill holls in the grid if smaller elements are added. Can be paired with row or column : "grid-auto-flow: row dense;")
order: (integer) default is 0

Value/Meaning

rem Same as em but relatively to <html> tag.
% Sets a non textual element (div, image, padding, margin...) size relatively to his parent element.
min/max-width/height: Generaly in px to ensure hard limits on the element's dimensions.
aspect-ratio: value / value
