Elements and Structure

Tags

Meaning
<!DOCTYPE html> Specifies the type of document and HTML version.
<html> Contains all HTML language of the page.
Added after the !DOCTYPE declaration.
<head> Contains general informations about the page that is not displayed.
<title> Defines the title of the HTML document, visible in the browser's title bar or tab.
<body> The visible content of an HTML document.
<h1>-<h6> Creates one of the 6 heading levels.
<div> Creates a container.
<p> Creates a paragraphe.
<ol> Creates an ordered list.
<ul> Creates an unordered list.
<li> Creates a list item.
<img> Adds an image.
Does not need to be closed.
<video> Creates a video media player for video playback.
<audio> Creates a video media player for video playback.
<a> Creates hyperlinks via the href attribute.
<link> Creates a link between the HTML file and an external resource (stylesheet, icon, font...).
<strong> Highlits text (generally appears bold).
<em> Emphasises text (generally italicize).
<br> Creates a line break.
<span> Generic INLINE container.
<!-- --> Adds an invisible comment.
<>

Attributes

Effects
id="" Adds a specific and UNIQUE attribute to an HTML element to be called in CSS and JS.
Can have multiple values
Valid id attributes should begin with a letter and should only contain letters (a-Z), digits (0-9), hyphens (-), underscores (_), and periods (.).
class="" Adds a specific and REUSABLE attribute to an HTML element to be called in CSS and JS.
Can have multiple values
Valid class attributes should begin with a letter and should only contain letters (a-Z), digits (0-9), hyphens (-), underscores (_), and periods (.).
href="" Specifies the adress of a link (CSS file, web page...).
Adding a "#" links it somewhere in the same page.
target="" "_blank" opens a link in a new tab.
"popup" opens a link in a new window.
src="" Specifies the location of an external source (image, video, audio, fichier JS...).
src="" Specifies the relationship between the current document and the linked document/resource.
alt="" Adds an alternative texte to an image
controls Adds controls to a video file
autoplay Plays at the loading of the page.
loop Plays in loop.
style="" Permits CSS specs in HTML
title="" Adds a tooltip when hovering a clickable element
width:/height: Sets width and height of a video.
poster="url" Sets the default image of an unlaunched video.

Tables

Tag

Meaning
<table> Insert a table
<thead> Insert a Table Header (Semantic)
<tbody> Insert a Table Body (Semantic)
<tfoot> Insert a Table Footer (Semantic)
<tr> Insert a row
<td> Insert data (and columns)
<th> Insert table heading
<caption> Adds a title to the table.

Attributes

Effects
rowspan="" Indicate how many rows this cell should span
colspan="" Indicate how many columns this cell should span

Forms

Tag

Meaning
<form> Creates and contains the form.
action= Determines where the action is sent.
method= Defines the method of sending the data ("post", "get")
<input> Adds an input field (doesn't need a closing tag).
type= Defines the type of input:
  • "text"
  • "password"
  • "number" (min="" max="" step=""...)
  • "range" (min="" max="" step=""...)
  • "checkbox" (checked)
  • "radio" (same name="" but different value="")
  • "submit" (sends the data)
  • "tel"
  • "email"
  • "url"
  • "color""
  • "date"
  • "search"
name= Identifies the form when sended back to the server.
value= What is typed into the text field.
step= Creates arrows to choose from the specified steps (if 1 then -1, 0, 1, 2... if 12 then -12, 0, 12, 24...)
<label> Explains what the <input> is used for.
for= Identifies which input is targeted
id= Links an input to a label (same as for="")
<select> Adds a dropdown menu.
<option> Adds an option inside the dropdown menu. Add selected if selected by default required.
<datalist> Adds a datalist.
list= Identifies the list inside the input.
id= Identifies the datalist
<textarea> Adds a bigger field of text (replaces <input>).
rows="" cols="" Specifies the number of rows and columns of the textarea.
required Specifies a required input.
autofocus Sets the cursor directly on the desired input.
min="" max="" Sets min and max values for "number" and "range".
minlenght="" maxlength="" Sets the min and max values for "text" and "textarea".
pattern= Sets rules for the types of characters that can be used.
placeholder="" Sets a temporary text.

Semantic HTML

Tag

Meaning
<header> Indicates a header.
<main> Indicates the main interest of the page.
<footer> Indicates a footer.
<nav> Indicates a navigation tool.
<aside> Indicates an aside section.
<section> Creates a section (generic).
<article> Indicates an article.
<figure> Indicates an image, diagram or code snippet.
<figcaption> Used inside the <figure> element, it creates a description for a figure.
