Data Types
  • Number (any number).
  • String (text inside "" or '').
  • Boolean (True or False).
  • Null (intentionnal absence of value).
  • Undifined (a given value does not exist).
  • Symbol (unique identifiers).
  • Object (collection of related data).
Operators
  • Add (+)
  • Substract (-)
  • Multiply (*)
  • Divide (/)
  • Remainder (%)
  • Dot operator (.) : used to access properties and methods.
Concatenation Appending one string to another (i.e. 'string'+'string'=stringstring).
Property Stored information of an object i.e. .lengthLogs reveals a string's number of caracters.

Built-in objects console, Math, Number...
Variable Container for value (labels and stores data in memory).
What to do with variables ?
  • Create a variable with a descriptive name.
  • Store or update information in a variable.
  • Reference or "get" information stored in a variable.
Keywords Tokens that have a special meaning, the list here.
var Declares or creates a new variable.
let Signals that the variable can be reassigned a different value.
const Signals that the variable cannot be reassigned a different value.
Mathematical assignment operators : 
let w = 4;
w = w + 1;
console.log(w); // Output: 5
can be written: 
let w = 4;
w += 1;
console.log(w); // Output: 5
as for: 
let x = 20;
x -= 5; // Can be written as x = x - 5
console.log(x); // Output: 15
 
let y = 50;
y *= 2; // Can be written as y = y * 2
console.log(y); // Output: 100
 
let z = 8;
z /= 2; // Can be written as z = z / 2
console.log(z); // Output: 4
