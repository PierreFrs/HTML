Introduction

Commands/Concepts Meaning
console.log() Used to log or print messages in the console.
//comment For single line comments.
/*comment*/ For multiple lines comments.
Data Types
  • Number (any number).
  • String (text inside "" or '').
  • Boolean (True or False).
  • Null (intentionnal absence of value).
  • Undifined (a given value does not exist).
  • Symbol (unique identifiers).
  • Object (collection of related data).
Operators
  • Add (+)
  • Substract (-)
  • Multiply (*)
  • Divide (/)
  • Remainder (%)
  • Dot operator (.) : used to access properties and methods.
Concatenation Appending one string to another (i.e. 'string'+'string'=stringstring).
Property Stored information of an object i.e. .lengthLogs reveals a string's number of caracters.
Methods Actions we can perform on an object : "exemple string".methodName().
For a list of these methodes see the MDN page
Built-in objects console, Math, Number...
Variable Container for value (labels and stores data in memory).
What to do with variables ?
  • Create a variable with a descriptive name.
  • Store or update information in a variable.
  • Reference or "get" information stored in a variable.
Keywords Tokens that have a special meaning, the list here.
var Declares or creates a new variable.
let Signals that the variable can be reassigned a different value.
const Signals that the variable cannot be reassigned a different value.
Mathematical assignment operators : 
let w = 4;
w = w + 1;
console.log(w); // Output: 5
can be written: 
let w = 4;
w += 1;
console.log(w); // Output: 5
as for: 
let x = 20;
x -= 5; // Can be written as x = x - 5
console.log(x); // Output: 15
 
let y = 50;
y *= 2; // Can be written as y = y * 2
console.log(y); // Output: 100
 
let z = 8;
z /= 2; // Can be written as z = z / 2
console.log(z); // Output: 4
Increment operator (++) and decrement operator (--) increase (++) or decrease (--) the value of a variable by 1.
Template litteral (string interpolation) (``) 
const myPet = 'armadillo';
console.log(`I own a pet ${myPet}.`);
// Output: I own a pet armadillo.
typeof Checks the data type: 
							const unknown1 = 'foo';
							console.log(typeof unknown1); // Output: string
							 
							const unknown2 = 10;
							console.log(typeof unknown2); // Output: number
							 
							const unknown3 = true; 
							console.log(typeof unknown3); // Output: boolean
String concatenation 
let service = 'credit card';
let month = 'May 30th'; 
let displayText = 'Your ' + service  + ' bill is due on ' +  month + '.';

console.log(displayText);
// Prints: Your credit card bill is due on May 30th.

Conditionnals

Concepts Meaning
Control flow Order in which statements are executed in a program, can be altered by control structures (i.e. conditionnals).
if(true or false) {} If true : execute, if false: block.
if...else {} Adds a default behavior when the statement is false.
Comparison statements
  • Lesser than : <
  • Greater than : >
  • Lesser or equal : <=
  • Greater or equal : >=
  • Equal : ===
  • Not equal : !==
Logical operators
  • The 'and' operator : &&
  • The 'or' operator : ||
  • The 'not' (or 'bang') operator : !
Thruthy values Values that evaluate to 'true' when checked as a condition.
Falsy values Values that evaluate to 'false' when checked as a condition (i.e. 0, "", '', Undifined, NaN...)
Ternary operation condition ? expression1 : expression2;
else if Allows to add multiple statement between if... and ...else (if 'return' then 'break' is superfluous).
i.e. :
const size = 10;

if (size > 100) {
  console.log('Big');
} else if (size > 20) {
  console.log('Medium');
} else if (size > 4) {
  console.log('Small');
} else {
  console.log('Tiny');
}
// Print: Small
switch (compared value) {
 case 'value1' :
  console.log();
  break;
 case 'value2' :
  console.log();
  break;
 default:
  console.log();
 }		 i.e; : 
let groceryItem = 'papaya';
 
switch (groceryItem) {
  case 'tomato':
    console.log('Tomatoes are $0.49');
    break;
  case 'lime':
    console.log('Limes are $1.49');
    break;
  case 'papaya':
    console.log('Papayas are $1.29');
    break;
  default:
    console.log('Invalid item');
    break;
}
 
// Prints 'Papayas are $1.29'

Scope

Concept Meaning
Scope Reffers to where the function is accessible within the program:
  • Global scope: can be found anywhere in the program.
  • File or module scope: can be find within the file.
  • Function scope: accessible within the function.
  • Code block scope: accissible within a {...} codeblock.
