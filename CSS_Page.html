Syntax and Selectors

Code

Meaning
<link href="" rel="" type=""> Links a CSS stylesheet to an HTML document.
<style> Allows to write CSS inside an HTML file.
style="" Allows inline CSS styling inside an HTML file.
any tag Type selector.
.stuff Class selector.
.stuff.stuff Selects a two word class.
#stuff ID selector.
stuff, stuff Used to group CSS selectors.
stuffstuff Chains multiple selectors (ex: h2.banner {} = selects the h2 types that have a .banner class).
ID>class>type Selectors hierarchy.
stuff stuff Descendant selector : matches the element that are descended from the previous selector (ex: p ol {} = only the ol that descend from p will be selected).
stuff + stuff Selects two adjacent siblings

Pseudo-class selectors

Meaning
:link Styling before visiting.
:visited Styling after visiting.
:hover Replace a rule by another one when hovering an element.
:active Styling while activating.
element:nth-child(integer, even, odd or formula) Selects the nth child element of the targeted type's parent.
element:nth-of-type(integer, even, odd or formula) Selects the nth element of the targeted type only.
::before or ::after Do an action before or after the selected element.

The Box Model

Declaration

Value/Meaning
padding:
  • -top, -right, -bottom, -left
  • px, em, %
  • inherit, initial, revert, revert-layer, unset
border:
  • width (thin, medium, thick, px, em, rem, %...)
  • style (none, dotted, inset, dashed...)
  • color
  • radius (px, em, %)
border-collapse: collapse (fusion of borders) or separate (default).
outline: The same as borders but don't take space.
margin: Same as padding + auto (horizontally centers an element).
width:, height: px, em, %...
max-/min- (width/height) px, em, %...
box-sizing: border-box
overflow: visible, hidden, scroll
visibility: hidden

Display and Positionning

Declaration

Value/Meaning
z-index: integer (+/-)
position:
  • static (default)
  • relative (from the top:, right:, bottom:, left: relatively to its position)
  • absolute (on the top:, right:, bottom:, left: of it's closest parent, out of flow)
  • fixed (top:, right:, bottom:, left: of the screen, out of the flow)
  • sticky (sticks to the screens when reaches position top:, right:, bottom:, left:, out of the flow)
display: inline, block, inline-block
float: top, left, right, bottom (width must be specified, out of flow)

Animations and Transitions

Declaration Value/Meaning
transition-property: States what property we want to transition.
transition-duration: Specifies the duration of the transition.
transition-timing-function: Describes the pace of the transition (https://developer.mozilla.org/en-US/docs/Web/CSS/transition-timing-function)
transition-delay: Specifies the time to wait before starting the transition.
transition: Shorthand for the four previous declarations (transition: property duration timing-function delay).
transition: all Selects all the properties that will transition.
https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_animated_properties List of all animated properties.
